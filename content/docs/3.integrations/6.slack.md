## Overview

Integrate Castrel with Slack to enable intelligent alert triage and root cause analysis directly from your team's communication channels.

## Setup Steps

### 1. Install from the Castrel Platform

Navigate to the **Integrations** page in the Castrel platform and click the **Connect** button for Slack.

![Integrations Page](/images/slack/slack-connection.jpg)

On the Slack Integration page, click the **Install to Slack Workspace** button to start the installation process.

![Install Slack Integration](/images/slack/slack-install.jpg)

### 2. Authorize Castrel in Slack

You will be redirected to Slack's authorization page. Select your Slack workspace and click **Allow** to grant Castrel the necessary permissions.

![Slack Authorization](/images/slack/slack-authorization.jpg)

### 3. Configure Channels

After authorization, you will be redirected to the **Slack Successfully Installed** page. Click the **Configure Channels** button in the upper right corner.

![Configure Channels](/images/slack/config-channel1.jpg)

In the configuration dialog, select the Slack channels for receiving alert notifications and incident investigation:

- **Alert Channel** - Channel for receiving system alert notifications.
- **Incident Channel** - Channel for incident investigation and AI analysis.

Click **Save Configuration** to complete the setup. Castrel will automatically join the selected channels. If automatic joining fails, please manually invite Castrel Bot in the Slack channel by typing `/invite @Castrel`.

![Configure Slack Channels](/images/slack/config-channel2.jpg)

## Features

Once integrated, Castrel provides the following capabilities:

### Alert Triage

Castrel monitors the configured channels for incoming messages. When an alert message is detected, Castrel automatically:

- **Classifies the alert** - Categorizes alerts by severity, type, and affected components.
- **Performs root cause analysis** - Investigates the underlying cause using connected observability data sources.

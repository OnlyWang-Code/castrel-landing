## Overview

- **Connector Name**: `github`
- **Supported API Version**: GitHub REST API v3
- **Official Documentation**: [GitHub REST API Docs](https://docs.github.com/en/rest)

## Available Tools

The connector provides a set of tools for repository analysis.

- **Commit Inspection**: Get commit details and diffs.
- **Repository Info**: Get repository metadata.
- **File Operations**: Read file contents and directory listings.
- **Issue/PR Management**: Search and manage issues and pull requests.
